# Building Automation Suite

Building Automation Suite is a powerful tool designed to streamline and automate tasks related to green building design, analysis, and management using gbXML files. This application provides a user-friendly interface to visualize, edit, and process building data with an emphasis on energy efficiency and environmental sustainability.

## Features

- **Visualize gbXML files**: Explore and analyze 3D building models with an intuitive user interface.
- **Edit and Clean gbXML files**: Fix errors and breaks in building geometry, ensuring accurate and reliable data.
- **Load Calculations**: Seamlessly integrate with Carrier HAP for load calculations and energy analysis.
- **Outside Air Calculations**: Utilize exported data to perform outside air calculations in Excel.

## Getting Started

### Prerequisites

- .NET Framework version 4.6.2
- Carrier HAP (optional for load calculations)
- Microsoft Excel (optional for outside air calculations)

### Installation

1. Download the latest release from the [Releases](https://github.com/username/BuildingAutomationSuite/releases) page.
2. Extract the zip file to your desired location.
3. Run the `BuildingAutomationSuite.exe` file to launch the application.

## Usage

1. Import a gbXML file using the *File > Open* menu or by dragging and dropping the file onto the application window.
2. Explore the 3D building model using the navigation controls provided.
3. Edit and clean the gbXML data as needed using the built-in tools.
4. Export the processed data for further analysis in Carrier HAP or Excel.
